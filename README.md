ISM 2411 Data Cleaning Project

This small project demonstrates a data cleaning pipeline for a messy sales CSV.

Run the cleaning script from the project root:

```bash
python src/data_cleaning.py
```

The script reads `data/raw/sales_data_raw.csv` and writes the cleaned output to
`data/processed/sales_data_clean.csv`.

Run the test suite (requires `pytest`):

```bash
pip install -r requirements.txt
pytest -q
```

Note: the repository intentionally includes a sample `data/raw/sales_data_raw.csv`.
The cleaned output `data/processed/sales_data_clean.csv` is generated by the
script. The `.gitignore` file excludes the processed CSV by default — if you
prefer the cleaned CSV to be tracked, remove it from `.gitignore`.

Files:
- `src/data_cleaning.py` — main script with documented cleaning steps.
- `data/raw/sales_data_raw.csv` — sample messy data (provided).
- `data/processed/sales_data_clean.csv` — generated by running the script.
- `reflection.md` — short reflection about using Copilot.
